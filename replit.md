# PHP Project

## Overview
A PHP web application running on the Replit platform using PHP 8.2's built-in development server.

## Architecture
- **Language**: PHP 8.2
- **Server**: PHP built-in development server (`php -S`)
- **Port**: 5000

- `index.php` - Main entry point (Hello World placeholder)

## Running the Project
The app starts automatically via the "Start application" workflow:
```

```
